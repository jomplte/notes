数组

在java中, 数组就是一个变量, 用于将相同数据类型的数据存储在内存中, 数组是在内存中划分一串连续的内存空间

创建数组时必须指定数据类型, 数据类型指定之后就是不可变的

数组的长度也必须指定, 指定后也不可变

数组中存放的数据叫做元素

数组中每个元素的位置叫做下标, 注意: 从0开始

定义数组

int[] array = {9,5,4,3,7,8};    // 直接指定了数组的类型/长度/元素的值

int array [] = {9,5,4,3,7,8};   // 不建议使用

int[] array = new int[10];  // 指定了数组的类型和长度

float[] array;
array = new float[8];   // 将数组array新初始化8个空间

array[2] = 200; // 为数组下标为2的元素赋值
System.out.println(array.length);   // 调用数组array的length属性, 获取数组的长度

各数据类型的数组默认值

  1. int数据类型的数组默认值为 0
  2. boolean数据类型的数组默认值为 false
  3. float和double数据类型的数组默认值为 0.0
  4. char数据类型的数组默认值为:

  1. 引用数据类型的默认值为 null
String s = '';   // ''不是null

循环

while循环

while (循环条件){
    循环体; // 循环执行循环条件结果为true时的循环体, 直到某次循环条件为false, 跳出循环
}
把0~100之间的偶数相加
public class C {
    public static void main(String[] args){
        int i = 0;
        int sum = 0;
        while (i < 100){
            i++;
            if(i == i >> 1 << 1){   
                sum += i;
            }
        }
        System.out.println(sum);
    }
}

public class D {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5,6,8};
        int sum = 0;
        int max = arr[0];
        int min = arr[0];
        float avg = 0;
        int i = 0;

        while (i < arr.length){
            max = max > arr[i] ? max : arr[i];
            min = min < arr[i] ? min : arr[i];
            sum += arr[i];
            i++;
            avg = sum * 1.0f / arr.length;
        }
        System.out.println("平均值是:" + avg);
        System.out.println("最大值是:" + max);
        System.out.println("最小值是:" + min);
    }
}

do...while循环

do {
    代码块; // 先执行一遍循环体, 然后进行条件的判断.如果条件为true, 则再次执行循环体
} while();

public class G {
    public static void main(String[] args) {
        int health = 100;
        int attack =13;
        int times = 0;
        do {
            times++;
            if (times % 5 == 0){
                health = health - attack * 2;   // 打怪物, 100血的怪物,每次攻击掉13滴血,每5次一次暴击
            } else {
                health = health - attack;
            }
        } while (health > 0);
        System.out.println("需要打怪的次数是" + times);
    }
}
