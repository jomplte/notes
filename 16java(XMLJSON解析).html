XML文件

  1. 什么是XML文件

    • 可扩展标记语言(EXtensible Markup Language);
    • 设计宗旨是传输数据, 而非显示数据;
    • 没有预定义标签, 需要自定义;
    • 纯文本文件;

  2. 为什么需要使用XML

    • 可以用来传输数据;
    • 以后学习的Spring框架/maven, 配置文件多数都是XML格式;
    • 为以后学习的HTML打下基础;
    • 学习解析, 方便学习以后的JS;

  3. 还有哪些数据格式的作用类似于XML

    • JSON, 目前绝大多数都用来传输数据. 少量的框架使用JSON格式的数据作为配置文件, 例如前端框架Angular2;
    • properties, 基本上都用作配置文件
    • yaml或yml, 比较新的一种文件格式, 简化properties, 用作配置文件.
    • 要不你就自定义

  4. XML格式文件的要点

    • 第一行必须为文档声明
    • 必须有根元素
    • 元素的标签是成对出现的, 必须有关闭标签
    • 元素的标签对大小写敏感
    • 元素之间必须正确的嵌套
    • 文档中的空格会被保留
    • 元素的属性值必须加""
    • 文档的注释使用<!-- 注释 -->形式, 注释不能写在属性中

  5. 特殊符号的引用

    XML代码 需替换的符号
    &lt; <
    &gt; >
    &amp; &
    &apos; '
    &quot; "
    <![CDATA[]]> 里面随便加, 快捷键CD

  6. XML文件示例

    <?xml version="1.0" encoding="utf-8" ?>

public class Main {
    public static void main(String[] args) throws DocumentException {
        // 1. 获取Document对象
        SAXReader reader = new SAXReader();
        // 1.1 read方法中的参数中可以填File/字节输入流/字节输入流等
        Document doc = reader.read(new File("album-list.xml"));
        // 2. 获取根元素
        Element root = doc.getRootElement();
        // 3. 获取某个元素的子元素的迭代器, 也可以root.element()然后使用增强for
        // 获取根元素albumList的子元素的迭代器
        Iterator<Element> iterator = root.elementIterator();
        // 遍历所有的album元素
        while (iterator.hasNext()) {
            Element albumEle = iterator.next();
            // 4. 获取某个元素的某个属性的值
            String albumName = albumEle.attributeValue("name");
            System.out.println(albumName);
            Iterator<Element> musicIterator = albumEle.elementIterator();
            while (musicIterator.hasNext()) {
                Element musicEle = musicIterator.next();
                // 5. 获取某个元素的子元素的文本, 这里可以用set存到对象里
                String id = musicEle.elementText("id");
                String name = musicEle.elementText("name");
                String duration = musicEle.elementText("duration");
            }
        }
    }
}

JSON 格式

JSON 格式的数据由两种数据类型组成
第一种: {} 对象
第二种: [] 数组或集合

对象中的数据都是以键值对(key-value)的形式保存的
key一定是字符串, value可以是数字/字符串/boolean/数组/null/对象

解析JSON数据常用三种框架:
1. jackson
2. gson 谷歌的
3. fastJson 阿里巴巴

新建文件libs gson-2.8.5.jar 右键选择Add as Library

public class JsonDemo {
    public static void main(String[] args) throws FileNotFoundException {
        // 解析步骤
        // 1. 创建gson对象
        Gson gson = new Gson();
        // 2. 判断json数据的最外层的数据类型, 确定解析的结果数据类型
        // 导包: java.lang.reflect.Type;
        Type t = new TypeToken<List<Album>>(){}.getType();
        // 3. 解析
        Reader r = new FileReader("album-list.json");
        List<Album> albumList = gson.fromJson(r, t);
        r.close();

    
        // 解析最外层是{}的情况
        // 1. 创建Gson
        // 2. 分析结果类型
        // 3. 解析
        Reader reader = new FileReader("album.json");
        Album album = gson.fromJson(reader, Album.class);
        reader.close();

        // 将对象序列化
        String json = gson.toJson(album);
        System.out.println(json);
    }
}

public class MainGoods {
    public static void main(String args[]) throws IOException {
        Reader r = new FileReader("snack-list.txt");
        BufferedReader br = new BufferedReader(r);
        List<Goods> goodsList = new ArrayList<>();

        Document d = DocumentHelper.createDocument();
        Element list = d.addElement("goodslist");

        String line = null;
        while ((line = br.readLine()) != null) {
            String[] s = line.split(",");
            Goods g = new Goods();
            g.setId(Integer.parseInt(s[0]));
            g.setName(s[1]);
            g.setPrice(Float.parseFloat(s[2]));
            g.setQuantity(Integer.parseInt(s[3]));
            g.setUnit(s[4]);
            goodsList.add(g);
            // 转XML
            Element gt = list.addElement("goods");
            gt.addAttribute("id", String.valueOf(g.getId()));
            gt.addElement("name").setText(g.getName());
            gt.addElement("price").setText(String.valueOf(g.getPrice()));
            gt.addElement("quantity").setText(String.valueOf(g.getQuantity()));
            gt.addElement("unit").setText(g.getUnit());
        }
        br.close();
        r.close();
        // 转JSON
        //Gson gson = new Gson();
        Gson gs = new GsonBuilder().setDateFormat("yyyy-MM-dd").create(); // 有date时,用这种
        String s = gson.toJson(goodsList);
        Writer w = new FileWriter("snack-list.json");
        w.write(s);
        w.close();

        Writer w1 = new FileWriter("snack-list.xml");
        d.write(w1);
        w1.close();
    }
}
