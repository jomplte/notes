二维数组

public class A {
    public static void main(String args[]){
        int[][] b = new int[4][3];  // 4个长度为3的三维数组(可看做四行三列的矩阵)
        b[0] = new int[4];  // 将第一个一维数组的长度变为4

        for (int i = 0; i < b.length; i++) {
            for (int j = 0; j < b[i].length; j++) { 
                System.out.print(b[i][j] + " ");    //二维数组的遍历
            }
            System.out.println();
        }
    }
}

public class A {
    public static void main(String[] args) {
        int[][] b = new int[][]{{1,2,3},{2,3,4},{3,4,5}};

面向对象

com.lanou3g.oop 包名是公司域名反写加项目名, 小写, 以后在创建工程的时候, 必须指定包名, 并且所有类必须都创建在该包的子包下, 不允许直接在src文件夹下创建类

package com.lanou3g.oop; 必须写在第一行

一个创建对象的过程

1. 声明变量名
     
2. 堆内存中划分内存空间
     
3. 在内存空间中给初始值

在类中直接声明的变量, 就是该类的属性, 属性的声明实际上是一种规范, 规范了该类的所有对象都有这几个属性(类声明了对象的共有属性)

新建类

新建个类就相当于新建了一个新的引用数据类型

新建对象

类名 变量名(对象名) = new 类名()

某个类中的所有对象都是独立的, 改变其中一个对象的状态, 不会对其他对象产生影响

当一个对象创建好之后, 会把该对象的所有属性都划分好,换句话说对象的属性都是有初始值的

属性的别名: 实例变量 成员变量

方法

访问权限修饰符 返回值类型 方法名(参数列表) {
       方法体;
   }

返回值: 代表该方法执行后的结果, 如果没有返回值, 该位置填 void 方法只有调用才会执行. 同一个方法可以调用多次

访问权限修饰符

可以修饰属性和方法, 用来规定某个属性或者方法的可访问范围

  1. private 只能在同类下访问
  2. 什么也不写, 默认的, 只能在同一个包下访问
  3. protected 受保护的, 同包其他类或子类有访问权限
  4. public 公开的, 在任何位置都可以访问
  5. Java规定，类的权限设定会约束类的成员上的权限设定
class AnyClass {
   public void doString() {
       // ...
   }
}

等同于 此时，即使AnyClass类中的doString()方法又被设置成public访问权限，其访问权限也与AnyClass类的访问权限相同。因为Java规定，类的权限设定会约束类的成员上的权限设定，所以上述代码等同于下面的代码：

class AnyClass {
    void doString() {
        // ...
    }
}
  1. 在以后的开发过程中, 所有的属性和方法的访问权限越小越好

            //为了防止出现不必要的空指针异常

    // 参数声明的是什么类型, 那么在方法的调用时就得填入什么类型
    // 方法的参数也称作 形参(形式参数)
    // 多个参数声明的是什么顺序, 那么调用时就得按什么顺序填入参数
    void eat(String food, int count) {
        System.out.println(name + "吃了" + food);

     * 用来向购物车中添加商品

    void add(Goods goods) {
        if (index < shoppingCart.length) {
            shoppingCart[index] = goods;
            index++;
        } else {
            Goods[] newCart = new Goods[shoppingCart.length];   // 新数组
            for (int i = 0; i < shoppingCart.length; i++) {
                newCart[i] = shoppingCart[i];
            }
                newCart[index] = goods;
                index++;
                shoppingCart = newCart;
        }
    }
    void clear(){
        shoppingCart = new Goods[shoppingCart.length];
        System.out.println("购物车是空的, 快去添加购物车吧");
    }
